# Soundcloud Desktop

A simple, lightweight, unbloated soundcloud app written in rust using tauri.


## Build/Install

To build the Soundcloud Desktop App:

- Make sure you have downloaded [the rust programming language](https://rustup.rs)
- Download [the rust tauri cli](https://v2.tauri.app) via cargo

Then run either of the following:
```bash
  cargo tauri build
```
Or run the build batch script
```bash
  .\build.bat
```

For a simple install, simply download the installer and follow its prompts.
## Whats included?

- Built in ad blocker
- Displays Discord activity (RPC)
- \>5mb build size, >2mb installer
- \>1% cpu usage, >5mb ram usage
